HTML is HyperText Markup Language

Used to create the structure of a webpage.

Markup language, not a programming language.

Extension: .html or .htm

Browser Role: Browsers read HTML files and render them as web pages.
HTML (HyperText Markup Language) is the standard language used to create and structure web pages. It defines the content and structure of a webpage, while CSS handles styling and JavaScript handles interactivity.
